AirPods Controller v2.1
Управление AirPods Pro на Windows

Что умеет программа:
• Переключение ANC / Прозрачность
• Горячие клавиши
• Авто-подключение при старте Windows
• Трей-иконка для быстрого доступа

Требования:
• Windows 10 / 11
• .NET 8 Desktop Runtime
• Тестовый режим Windows (testsigning)
• Отключенный Secure Boot в BIOS

Если после установки программа не видит наушники —
проверьте, установлен ли драйвер в Диспетчере устройств.
